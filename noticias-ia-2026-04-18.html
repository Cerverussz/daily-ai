🔥 TOOLS OpenAI Codex Desktop: computer use en background + 90 plugins nuevos

Codex Desktop ahora permite que agentes controlen tu Mac en background — ven pantalla, hacen clic y escriben con su propio cursor sin interferir con tu trabajo. Incluye browser in-app con comentarios inline y más de 90 plugins nuevos: Atlassian Rovo, GitLab Issues, CircleCI, CodeRabbit, Microsoft Suite y Render, entre otros.

Agentes trabajando en paralelo en tu máquina sin interrumpirte es el siguiente paso natural — automatiza tareas repetitivas de DevOps mientras sigues escribiendo código.

🔥 ANDROID Google AI Edge Gallery: function calling on-device con FunctionGemma

Google lanzó on-device function calling en AI Edge Gallery con FunctionGemma, un modelo de 270M parámetros diseñado específicamente para tool calling. En Pixel 7 Pro alcanza 1,916 tok/s en prefill y 142 en decode. Agent Skills permite agregar capacidades modulares al modelo sin inflar el system prompt.

Un modelo de 270M parámetros haciendo function calling a ~2K tok/s en un Pixel 7 abre la puerta a agentes offline reales en apps Android — sin latencia de red, sin costos de API.

Fuente: Google Developers Blog · GitHub

ANTHROPIC Claude Managed Agents en public beta: agentes via REST API

Anthropic lanzó la public beta de Managed Agents con endpoints REST (/v1/agents, /v1/environments, /v1/sessions) que empaquetan agent loop + tool execution + sandbox + persistencia de estado. Se declaran MCP Servers directamente en la definición del agente, y los Skill packages existentes se integran sin modificaciones.

Montar un agente de Claude en producción pasa de semanas de infra custom a una llamada API — ideal si eres founder técnico construyendo sin equipo de DevOps.

🔥 ANDROID Gemma 4 optimizada para Arm: menos de 1.5GB para IA generativa mobile

Arm publicó guías de optimización para Gemma 4 E2B en dispositivos móviles. Con cuantización de 2-bit y 4-bit más embeddings memory-mapped por capa, el modelo corre con menos de 1.5GB de RAM vía LiteRT-LM. Compatible con CPU/GPU en Android e iOS, con aceleración KleidiAI para menor latencia y consumo energético.

Gemma 4 E2B con 1.5GB de RAM significa que puedes integrar IA generativa en apps Android sin matar la experiencia del usuario en dispositivos mid-range.

Fuente: Arm Newsroom · Google Developers

🔥 MOBILE ExecuTorch 1.0 GA: PyTorch on-device listo para producción

Meta y Arm lanzaron ExecuTorch 1.0 en general availability — el runtime oficial de PyTorch para IA on-device. Soporta desde microcontroladores hasta smartphones flagship con Arm CPUs, GPUs y Ethos-U NPUs. Las optimizaciones KleidiAI reducen latencia y memoria, y el nuevo backend VGF soportará GPUs Arm 2026 para rendering con ML.

Si tu app ya usa PyTorch para ML, ExecuTorch 1.0 es el camino directo a producción on-device sin reescribir modelos — el equivalente mobile de "just deploy".

Fuente: PyTorch Blog · Arm Newsroom
