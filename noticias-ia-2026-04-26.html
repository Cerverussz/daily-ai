🔥 DeepSeek V4 ya disponible: open-source con 1M tokens de contexto y precios agresivos

GPT-5.5 de OpenAI llega a la API con enfoque en agentes autónomos

OpenAI lanzó GPT-5.5 el 23 de abril, disponible en API desde el 24. Logra 88.7% en SWE-bench y reduce alucinaciones un 60% vs GPT-5.4. Su fuerza principal: completa tareas multi-paso planificando, usando herramientas, y autocorrigiéndose antes de entregar resultados.

🔥 Microsoft lanza Agent Governance Toolkit: seguridad runtime para agentes AI, open-source

Toolkit open-source (MIT) de 7 paquetes (Python, TypeScript, Rust, Go, .NET) que intercepta cada acción de un agente AI antes de ejecutarla con latencia <0.1ms. Es el primero en cubrir los 10 riesgos OWASP para agentes AI, con integraciones nativas para LangChain, CrewAI, Google ADK y OpenAI Agents SDK.

Si estás construyendo agentes autónomos para tu marketplace o plataforma, esto resuelve el problema de gobernanza que antes tenías que armar a mano.
