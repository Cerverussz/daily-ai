🔥 Google lanza Gemini Enterprise Agent Platform: Vertex AI evoluciona hacia agentes autónomos

Google Cloud rebautizó Vertex AI como Gemini Enterprise Agent Platform, una plataforma end-to-end para construir, escalar y gobernar agentes AI que ejecutan workflows multi-paso de forma autónoma. Incluye Agent Designer (no-code), soporte para agentes de larga duración, Inbox para monitorear actividad de agentes, y acceso a 200+ modelos vía Model Garden (Gemini 3.1 Pro, Claude, Llama, Gemma 4).

Si usas GCP, esta es la señal de que los agentes autónomos son ahora first-class citizens en tu infra — el Agent Designer no-code puede acelerar prototipos de automatización en tu marketplace.

Innovaciones técnicas con potencial práctico

🔥 Mistral Medium 3.5: open weights, 128B denso, self-hostable en 4 GPUs + Vibe Remote Agents

Mistral lanzó Medium 3.5, un modelo denso de 128B parámetros con 256K de contexto y licencia MIT modificada en Hugging Face. Logra 77.6% en SWE-Bench Verified y se ejecuta en solo 4 GPUs. Junto al modelo, presentaron Vibe Remote Agents: coding agents asincrónicos que corren en la nube y generan pull requests completos en GitHub desde CLI o Le Chat.

A $1.5/M input, con open weights y self-hosting real, es la alternativa más seria para pipelines de coding en tu propia infra sin depender de APIs americanas.

🔥 NVIDIA Nemotron 3 Nano Omni: modelo multimodal para edge con 3B parámetros activos

NVIDIA liberó Nemotron 3 Nano Omni, un modelo MoE de 30B parámetros (3B activos por token) que unifica visión, audio y texto con contexto de 256K tokens. Corre en 25GB de RAM, ofrece hasta 9x más throughput que otros modelos omni abiertos, y está optimizado para inferencia on-device y baja latencia.

Con 3B activos y 25GB de RAM, esto cabe en hardware de edge real — abre la puerta a features multimodales offline en apps móviles o dispositivos embebidos sin cloud.

OpenAI alerta: actualizar apps macOS urgente tras supply chain attack vía Axios

OpenAI emitió una alerta urgente requiriendo que todos los usuarios macOS actualicen ChatGPT Desktop, Codex y Atlas antes del 8 de mayo. Una versión comprometida de la librería Axios (v1.14.1) inyectó un RAT en el workflow de firma de código. OpenAI rotó certificados y publicó builds nuevos.

Creación de negocios y productos con IA

🔥 Mistral Vibe Remote Agents: coding agents async que generan PRs completos desde CLI

Junto con Medium 3.5, Mistral lanzó Vibe Remote Agents: agentes de coding que corren de forma asincrónica en la nube, se lanzan desde CLI o Le Chat, y producen pull requests terminados en GitHub. El flujo es: describes la tarea, el agente trabaja en background, y recibes un PR listo para review.

Para un founder técnico que trabaja solo o con equipo pequeño, delegar tareas de código a agentes async mientras haces otras cosas es un multiplicador de productividad real.

🔥 Claude Cowork: Live Artifacts con datos en tiempo real conectados a tus apps

Anthropic lanzó Live Artifacts en Claude Cowork: dashboards y trackers que se conectan a tus apps y archivos, y se refrescan automáticamente con datos actuales cada vez que los abres. Puedes compartir artifacts AI-powered con otros usuarios sin necesidad de API keys ni costos adicionales.

Claude Security entra en beta pública: escaneo de vulnerabilidades con Opus 4.7

Anthropic lanzó Claude Security en beta pública para clientes Enterprise. La herramienta escanea codebases completas razonando como un investigador de ciberseguridad: traza flujos de datos, examina interacciones entre componentes, y genera reportes con impacto, pasos de reproducción y fix recomendado.
