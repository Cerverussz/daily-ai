ANTHROPIC Claude Opus 4.7 disponible: mejor en código, visión y tareas largas

Anthropic lanzó Claude Opus 4.7 como su modelo más capaz en general availability. Mejora significativa en software engineering, tareas de codificación largas y complejas, visión de alta resolución, y añade controles de esfuerzo (effort controls) y task budgets — al mismo precio que Opus 4.6.

Si usas Claude Code para tu producto, el upgrade es gratis y el salto en tareas de SWE multi-archivo es notable — menos iteraciones para llegar al resultado correcto.

Fuente: CNBC · Anthropic Blog

🔥 ANTHROPIC Claude Code Routines: automatización cloud sin sesión activa

Routines permite configurar un prompt + repo + conectores que se ejecutan automáticamente por schedule (cron), API call o evento de GitHub (nuevo PR, push, etc.) desde infraestructura cloud de Anthropic. Pro: 5/día, Max: 15/día, Team/Enterprise: 25/día.

Esto convierte a Claude Code en un CI/CD inteligente — puedes automatizar triage de issues, verificación post-deploy o detección de drift en docs sin tener tu máquina encendida.

🔥 TOOLS Roblox Studio se vuelve agéntico: Planning Mode + playtesting automático

Roblox renovó su Assistant con Planning Mode: analiza código y data model del juego, hace preguntas, genera un plan editable y lo ejecuta con herramientas de mesh generation y modelos 3D procedurales. El patrón "planificar → ejecutar → testear → corregir" de forma autónoma es exactamente lo que todo AI coding tool debería copiar — Roblox lo implementó end-to-end para game dev.
