🔥 DoorDash lanza suite AI para merchants: onboarding 35% más rápido, fotos AI y web builder

DoorDash presentó herramientas AI que permiten a merchants lanzar su presencia online en minutos: el onboarding con AI extrae datos de su web existente (fotos, horarios, menú), el editor de fotos ofrece AI Retouch, AI Replate y Match the Style sin alterar la comida, y el Commerce Platform genera websites branded con AI que logran ~10% de conversión promedio en tests.

Es el playbook concreto de cómo AI reduce fricción en onboarding de un marketplace — directamente aplicable si construyes una plataforma con proveedores.

IBM Think 2026: watsonx Orchestrate se convierte en control plane para agentes AI multi-source

IBM anunció la nueva generación de watsonx Orchestrate (private preview), un control plane agéntico donde puedes desplegar agentes de cualquier fuente con políticas de gobernanza consistentes. También lanzó IBM Bob (GA), un partner de desarrollo agéntico para enterprise con seguridad y control de costos integrados, y Concert Secure Coder que embebe gestión de seguridad directamente en VS Code.

Si tu producto orquesta agentes de múltiples proveedores, el pattern de "control plane + policy enforcement" que IBM formaliza es el que necesitas implementar.

🔥 Saperly: el primer carrier telefónico construido para agentes AI con MCP nativo

Saperly es un carrier que da a tus agentes AI un número telefónico real con voz, SMS, routing y compliance listos en segundos. Ofrece un servidor MCP que permite a cualquier agente compatible (Claude Code, Cursor, Codex) hacer y recibir llamadas con caller ID persistente.

Si estás construyendo agentes que necesitan comunicarse con usuarios por teléfono (soporte, ventas, scheduling), esto elimina semanas de integración con Twilio y compliance.

🔥 Unity AI entra en Open Beta: asistente AI integrado en el editor Unity 6

Unity lanzó Unity AI en open beta para todos los developers en Unity 6. Incluye un asistente in-editor que escribe C# scripts, genera escenas desde imágenes, crea assets placeholder (materiales, sonidos, cubemaps, 2D/3D), con Plan Mode y rollback de cambios generados por AI. Si haces juegos o experiencias interactivas, esto baja dramáticamente la barrera para prototipar — y el modelo de $10/mes para Personal lo hace accesible para indie devs.

Claude y ecosistema Anthropic

Claude Code mayo: tokenizer mejorado, /model picker para gateways, y MCP auto-retry

Opus 4.7 recibió un nuevo tokenizer que mejora comprensión de inputs. El /model picker ahora lista modelos desde el endpoint /v1/models de tu gateway cuando usas ANTHROPIC_BASE_URL. Se añadió "claude project purge" para limpiar estado, MCP servers con errores transitorios hacen auto-retry (hasta 3 veces), y se corrigieron leaks de memoria de ~2GB en transcripts largos.

El auto-retry de MCP y el soporte de gateways custom hacen Claude Code más viable para equipos con infra propia.

🔥 Fitbit AI Coach en public beta: Gemini integrado con coaching proactivo en tiempo real
