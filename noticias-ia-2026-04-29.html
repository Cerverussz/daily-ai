Mistral AI lanza Workflows: orquestación enterprise con Temporal y human-in-the-loop

Mistral liberó Workflows en public preview dentro de Studio: defines pipelines AI multi-paso en Python combinando modelos, agentes y conectores externos. Innovaciones técnicas con potencial práctico

🔥 ICLR 2026: "The Reasoning Trap" — modelos que razonan mejor alucinan más en tool calls

Un paper presentado en ICLR 2026 en Río demuestra que entrenar modelos con RL para razonar mejor aumenta las alucinaciones en llamadas a herramientas, no las reduce. OpenAI confirmó que o3 alucina en 33% de queries (vs. 16% de o1). Ni prompt engineering ni DPO cierran el gap de confiabilidad.

Si estás construyendo agentes que llaman APIs o tools, no asumas que un modelo más inteligente = menos errores en tool use — necesitas validación explícita en cada paso.

OpenReview (ICLR) • Asanify AI Digest

🔥 Gemini CLI v0.38: AI coding en terminal gratis con 1,000 requests/día y memoria entre sesiones

Google actualizó Gemini CLI con Gemini 3.1 Pro por defecto, soporte MCP nativo, y un servicio de memoria que extrae patrones de sesiones anteriores. El free tier ofrece 1,000 requests/día — suficiente para usarlo como coding assistant principal sin pensar en límites.

Claude Code se actualiza: /resume por PR URL, búsqueda de skills y mejoras MCP

La última actualización de Claude Code permite pegar una URL de PR en /resume para encontrar la sesión que la creó (GitHub, GitLab, Bitbucket), agrega búsqueda type-to-filter en /skills, y mejora velocidad de MCP y plugins.
