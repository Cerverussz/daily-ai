🔥 ANTHROPIC Claude Code Desktop rediseñado: sesiones paralelas, terminal integrado y Routines

Anthropic lanzó un rediseño completo de Claude Code Desktop con sidebar para múltiples sesiones simultáneas, terminal integrado, editor de archivos in-app, diff viewer optimizado para changesets grandes, y layout drag-and-drop. Además, lanzó Routines en research preview: automatizaciones que corren en la infra de Anthropic sin necesidad de laptop encendida, con triggers por schedule, API o eventos de GitHub.

Routines convierte a Claude Code de herramienta de coding a plataforma de automatización — un founder solo puede tener triage de bugs, review de PRs y deploys corriendo 24/7 sin infra propia.

Fuente: Claude Blog · 9to5Mac

ANTHROPIC Claude Design: prototipado visual directo desde conversación

Anthropic lanzó Claude Design, un producto experimental que permite crear prototipos, slides, one-pagers y visualizaciones directamente desde chat. Potenciado por Claude Opus 4.7, disponible en research preview para suscriptores Pro, Max, Team y Enterprise.
