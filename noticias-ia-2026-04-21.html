🔥 OpenAI lanza SDK de Agentes con sandbox nativo y soporte MCP

OpenAI actualizó su Agents SDK con entornos sandbox integrados que permiten a los agentes inspeccionar archivos, ejecutar comandos y editar código de forma segura. Claude Opus 4.7 ya disponible: SWE-bench 87.6%, visión 2576px y task budgets

Anthropic lanzó Claude Opus 4.7 el 16 de abril, con mejoras significativas en ingeniería de software, tareas de código complejas y de larga duración. Introduce task budgets para controlar el consumo de tokens en loops agénticos y soporte de imágenes en alta resolución (2576px / 3.75MP).

87.6% en SWE-bench y task budgets significan agentes de código más precisos y predecibles en costos — pricing sin cambios en $5/$25 por millón de tokens.

🔥 Cursor lanzó Composer 2: modelo propio de coding con 73.7 en SWE-bench Multilingual

Cursor Composer 2, construido sobre Moonshot AI Kimi K2.5 con entrenamiento extensivo y RL a gran escala, logra 61.3 en CursorBench (37% mejor que v1.5) y 73.7 en SWE-bench Multilingual. Se posiciona como modelo frontier de coding a $0.50 por millón de input tokens.

Un IDE con su propio modelo frontier optimizado para código a $0.50/M tokens cambia la ecuación — si tu workflow es Cursor + Claude Code, ahora tienes dos motores especializados complementarios.

Arduino Ventuno Q: Edge AI con 40 TOPS a menos de $300, desafiando a NVIDIA Jetson

Arduino y Qualcomm lanzaron Ventuno Q, una SBC con procesador Dragonwing IQ8 (NPU de 40 TOPS), 16GB RAM y microcontrolador STM32H5 para control en tiempo real. Une cómputo AI de alto rendimiento con control determinístico de actuadores en una sola placa.

Si estás prototipando productos con AI on-device (robots, kioscos, dispositivos de fitness), esta placa a menos de $300 baja dramáticamente la barrera de entrada vs. Jetson.

Novedades de Claude y ecosistema Anthropic

Anthropic prueba remover Claude Code del plan Pro ($20/mes) — solo 2% de signups afectados

El 21 de abril, Anthropic removió silenciosamente Claude Code de las páginas de pricing del plan Pro, moviéndolo como exclusivo del plan Max ($100/mes). Tras la confusión pública, aclararon que es un test con solo 2% de nuevos signups "prosumer" y que suscriptores existentes no están afectados.

Si tu workflow productivo depende de Claude Code, vigila este movimiento — si se concreta, $100/mes es el nuevo piso para AI coding integrado con Claude.
